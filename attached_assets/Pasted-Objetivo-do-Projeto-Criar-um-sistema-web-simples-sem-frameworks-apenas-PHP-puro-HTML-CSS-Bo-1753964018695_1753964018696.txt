Objetivo do Projeto:
Criar um sistema web simples (sem frameworks, apenas PHP puro + HTML + CSS + Bootstrap) que será o Clube de Vantagens da ANETI, com foco em apresentar empresas parceiras e permitir que membros da ANETI possam gerar cupons de desconto exclusivos.

🎯 Funcionalidades obrigatórias:
🔓 Área Pública:
Página inicial com:

Destaque de empresas em promoção

Campo de busca por nome, cidade, categoria ou palavra-chave

Lista de empresas parceiras com visual amigável (cartões ou lista)

Página de detalhes da empresa:

Nome, logo, categoria, cidade/estado

Descrição e regras do benefício

Botão "Gerar Cupom" (apenas se logado)

🧑‍💼 Área do Membro ANETI:
Tela de login (usuário e senha ou apenas e-mail fictício como placeholder por enquanto)

Após login:

Exibe nome, plano (Júnior, Pleno, Sênior)

Histórico de cupons gerados

Botão para gerar novo cupom ao acessar uma empresa

🎟️ Cupom de desconto:
Gerado via botão "Gerar Cupom"

Deve conter:

Nome da empresa

Código do cupom (pode ser random ou UUID)

Nome do associado

Data e hora

Cupom deve ser visualizado em tela + opção de baixar (em PDF simples ou HTML estilizado para impressão)

🏢 Área da Empresa Parceira (não precisa login agora):
Formulário de cadastro com os campos:

Nome, CNPJ, logo (upload), cidade/estado, e-mail, telefone

Categoria

Texto da oferta/benefício

Regras do cupom

Após cadastro, vai para aprovação manual (admin)

🛠️ Área Administrativa da ANETI:
Tela de login administrativo

Funções:

Listar empresas cadastradas e aprovar/recusar

Ver cupons gerados por membro e por empresa

Criar/editar categorias (ex: alimentação, tecnologia, educação etc.)

📦 Requisitos Técnicos:
Frontend: HTML5, CSS3, Bootstrap 5 (sem frameworks JS)

Backend: PHP puro (sem frameworks)

Banco de dados: MySQL

Upload de imagens (logo da empresa)

Organização do código por pastas:

/public – página inicial, login, detalhes de empresa

/admin – painel da ANETI

/empresa – painel da empresa

/includes – conexões, funções, etc.

/assets – imagens, css

Estilo clean e moderno, alinhado à identidade da ANETI (usar tons de azul #012d6a, branco e cinza)

📌 Extras (se possível):
QR Code no cupom gerado (redirecionando para a validação online do cupom)

Sistema de autenticação fictício por enquanto (sem envio de e-mails)
